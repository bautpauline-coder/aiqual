## 🎯 À propos d'AIQual

**AIQual** est une startup genevoise spécialisée dans le déploiement d'**agents IA** pour automatiser les processus métiers et accompagner les organisations dans leur transformation numérique.

Nous intervenons à l'intersection de :
- 🧠 **L'intelligence artificielle agentique** — conception et déploiement d'agents IA autonomes
- ⚙️ **L'automatisation de processus** — RPA augmenté par l'IA, orchestration de workflows
- 🔬 **La veille technologique** — suivi des dernières avancées en IA générative et LLMs
- 🎓 **Le consulting & la formation** — accompagnement des Business Specialists IA (en lien avec l'ESM Genève)

---

## 🚀 Ce que nous faisons

| Domaine | Description |
|---|---|
| **Agents IA** | Conception d'agents autonomes capables de raisonner, décider et agir |
| **Automatisation** | Automatisation de processus complexes via des pipelines IA |
| **Veille IA** | Curation et analyse des innovations technologiques du secteur |
| **Consulting** | Accompagnement stratégique pour l'adoption de l'IA en entreprise |
| **Formation** | Programmes dédiés aux Business Specialists IA — partenaire ESM Genève |

## 🛠️ Technologies utilisées

- **LLMs & Modèles** : OpenAI GPT-4, Anthropic Claude, Mistral, LLaMA
- **Frameworks agents** : LangChain, LangGraph, AutoGen, CrewAI
- **Orchestration** : n8n, Make, Zapier, Airflow
- **Infrastructure** : Python, FastAPI, Docker, GitHub Actions
- **Données** : RAG, bases vectorielles (Pinecone, Weaviate, Chroma)

## 📦 Installation

```bash
# Cloner le dépôt
git clone https://github.com/aiqual/aiqual.git
cd aiqual

# Installer les dépendances
pip install -r requirements.txt

# Configurer les variables d'environnement
cp .env.example .env
```
